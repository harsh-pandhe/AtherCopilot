# Contributing to Aether Co-Pilot

Thank you for your interest in contributing to **Aether Co-Pilot**! We welcome contributions of all kinds, from bug fixes and feature enhancements to documentation and design improvements.

## How to Contribute

### 1. Feature Requests & Bug Reports
- Check the [Issues](https://github.com/harsh-pandhe/AtherCopilot/issues) page to see if your idea or bug has already been reported.
- If not, open a new issue with a clear title and description.

### 2. Pull Requests
1. Fork the repository.
2. Create a new branch for your feature or fix (`git checkout -b feature/awesome-feature`).
3. Commit your changes with descriptive messages.
4. Push to your fork and submit a Pull Request.

## Development Setup

1. Clone your fork.
2. Install dependencies: `npm install`.
3. Create a `.env.local` file based on the README instructions.
4. Run the development server: `npm run dev`.
