This document outlines the steps required to set up and run Aether Co-Pilot on your local development environment.

\section{Prerequisites}
Ensure you have the following installed on your system:
\begin{itemize}
    \item \textbf{Node.js}: Version 18.0.0 or higher
    \item \textbf{npm}: Version 9.0.0 or higher
    \item \textbf{Firebase CLI}: \texttt{npm install -g firebase-tools}
    \item \textbf{Genkit CLI}: \texttt{npm install -g genkit}
\end{itemize}

\section{Setup Steps}

\subsection{1. Clone the Repository}
\begin{lstlisting}[language=bash]
git clone https://github.com/harsh-pandhe/AtherCopilot.git
cd AtherCopilot
\end{lstlisting}

\subsection{2. Install Dependencies}
\begin{lstlisting}[language=bash]
npm install
\end{lstlisting}

\subsection{3. Environment Configuration}
Create a \texttt{.env.local} file in the root directory with necessary credentials for Clerk, Google Gemini, and Firebase.

\subsection{4. Firebase Setup}
\begin{lstlisting}[language=bash]
firebase login
firebase init
\end{lstlisting}

\subsection{5. Running the Application}
\begin{lstlisting}[language=bash]
npm run dev
\end{lstlisting}
Access the app at \url{http://localhost:3000}.

\section{Genkit UI}
For testing AI flows in isolation:
\begin{lstlisting}[language=bash]
genkit ui
\end{lstlisting}
